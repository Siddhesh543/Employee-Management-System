# Employee Management System

A console-based **Employee Management System** developed using **Java**, **Hibernate ORM**, and **MySQL**.  
It provides complete **CRUD operations** for managing employees, departments, designations, attendance records, and salary processing ‚Äî perfect for learning Java backend development, ORM concepts, and layered architecture (DAO + Service + Presentation).

## Features

- **Employee Management**  
  - Add new employee  
  - View employee by ID  
  - View all employees  
  - Update employee details (phone, salary, etc.)  
  - Delete employee  
  - Search employees by name  

- **Department Management**  
  - Add department  
  - View all departments  
  - Delete department  

- **Designation Management**  
  - Add designation  
  - View all designations  
  - Delete designation  

- **Attendance Management**  
  - Mark daily attendance  
  - (Basic view support through service layer)  

- **Salary Management**  
  - Process monthly salary  
  - Automatic net salary calculation (basic + allowances + bonus - deductions)  
  - View salary records  

- **Reports**  
  - Total count of employees, departments, designations  

- Clean console UI with formatted tables  
- Input validation & error handling  
- Email uniqueness check  
- Hibernate automatic schema generation (`hbm2ddl.auto = update`)  
- Proper transaction management  

## Technologies Used

- Java (JDK 8 or higher)  
- Hibernate ORM (JPA annotations)  
- MySQL Database  
- Maven / manual JAR management (hibernate-core, mysql-connector-java)  
- Scanner (console input)  
- Layered architecture:  
  - Model (Entities)  
  - DAO (Data Access Objects)  
  - Service (Business Logic)  
  - Presentation (Console UI in `App.java`)
